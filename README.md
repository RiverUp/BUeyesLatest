
```
npm run build
```

### Run your unit tests
```
npm run test:unit
```

### Lints and fixes files
```
npm run lint
```
## 需要先安装一些插件  
### element-plus
```
npm install element-plus --save
```
### element-plus icon
```
npm install @element-plus/icons-vue
```

### axios
```
npm install axios
```
### postcss 移动端屏幕自适应
```
npm install postcss-px-to-viewport --save-dev
```
